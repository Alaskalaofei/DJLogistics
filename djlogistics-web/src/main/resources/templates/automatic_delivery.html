1、唯一走货路线 ( 有效排货 )

根据指定路线或目的地，只有唯一的一条走货路线

   2、唯一价格路线 ( 有效排货 )

在第1点的基础上，如有多条走货路线时，只有一条走货路线有有效价格

（此点只有在指定路线的比价排货下方可有效）

   3、价格最优路线 ( 有效排货 )

在第1点的基础上，如有多条走货路线时，价格最优的走货路线为价格最优路线（如价格计算为0除
外）（系统以当前时间作为报价提取时间,此点只有在指定路线的比价排货下方可有效）

   4、重量优先路线 ( 有效排货 )

在第2点的基础上，当包装类型为WPX并且在收货时有录入单件信息时，如果走货路线的材积除以
数 与指定路线的材积除以数不相同时，将重新计算出货重。如果此走货路线为价格最优的路线，为
重 量最优路线 （此点只有在指定路线的比价排货下方可有效）

   5、无有效价格 ( 有效排货 )

在第2点的基础上，如多条走货路线计算价格都为0，为无有效价格

   6、无走货路线 ( 有效排货 )

根据指定路线或目的地，没有对应的走货路线，为无走货路线；

如指定路线有多条走货路线，但未启用比价排货时，也属于无走货路线
